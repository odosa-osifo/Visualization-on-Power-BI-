<h1>Visualizations</h1>


<h2>Description</h2>
Project consists of an interactive dashboard for PQPLUS. it provides actionable insights as it analyzes the revenue generated by the platform on a daily, monthly and yearly basis as well as tracking candidates registration and activation
<br />


<h2>Softwares and Utilities Used</h2>

- <b>Microsoft Power BI</b> 
- <b>Trello(PQPLUS)</b>

<h2>Environments Used </h2>

- <b>Windows 10</b> (21H2)
